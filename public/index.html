About the Project

Sentiment Analysis of Comments Received Through E-Consultation

Government and healthcare bodies collect thousands of citizen comments through E-Consultation modules. Manually reading them is slow and inconsistent. This project applies Natural Language Processing to automatically gauge public sentiment, helping decision-makers respond faster and more fairly.

  • Automated classification of every submitted comment
  • Confidence scoring for transparent results
  • Built to connect with a Spring Boot + MySQL backend
